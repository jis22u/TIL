< Java만을 활용한 영상 목록 화면과 리뷰 관리 프로그램 >

### 목표
1. 객체지향 개념과 Java 프로그램의 기본 구조 및 흐름을 이해하고 활용할 수 있다.
2. Collection, IO API를 이용하여 데이터를 관리하는 프로그램을 작성할 수 있다.
3. JSON 데이터를 이용하여 프로그램에 적용할 수 있다.

### 기능
1. JSON 데이터를 파싱 처리한 후 영상정보를 제공하는 목록 화면과 영상에 대한 리뷰를 관리하는 프로그램을 구현한다.
2. 영상에 대한 리뷰 관리 프로그램은 영상목록화면에서 영상을 선택한 후 선택한 영상과 관련된 리뷰 목록, 등록 기능을 제공한다.
3. 회원가입 및 가입한 회원 정보를 보여주는 목록 화면을 구현한다.
4. 프로그램을 종료 시 가입한 회원의 정보를 파일로 저장하고 프로그램 시작 시 파일에 저장되어 있는 회원의 정보를 로딩한다.

### 보완점
1. 리뷰 관련 기능을 실행하고서 프로그램 종료를 위해 '0'을 눌렀지만 종료가 제대로 되지 않는다. 리뷰 관련 ui나 dao에서 종료를 위한 로직을 고려하지 않은 채 구현해서인 것 같다.
